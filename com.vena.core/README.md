# com.vena.core

Core ECS-style primitives for the Vena framework.

**Namespace:** `Vena`

Includes: World, Actor, Component, System, Controller, Archetype; FastList, SafeMap, NonAllocLinkList, LinkedStack; object pool (PoolObject, WeakArray, WeakList, WeakMap, WeakHashSet); GcWatch, ProfilerWatch, TimeWatch; FlowGraph (TaskGraph, TaskTrack, TaskClip, TaskEvent, Context); HierarchicalFsm, JobBalancer.
